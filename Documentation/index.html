Markup Markup customization and highlights

Heads up! It is recommended to choose your desired color schemes and layout before making any editing!

This template has 3 different home layout and a portfolio details page, it is responsive with additional two different color schemes. Every different section is wrapped within a section element with an unique id, and a 960px wide container with .container for the contents.

animate.css Contains CSS3 animation effects (by Daniel Eden).

JavaScript JavaScript customization and highlights

JavaScript Files

There are 8 Javascript files in the download. They are located in "js" folder.

Tag Description
jquery.backgroundvideo.min.js This script is used to make usuage of the home-video version easier.
jquery.cycle2.js jQuery Cycle2 plugin is used to create sliders (eg. testimonials slider)
jquery.textillate.js This plugin is used for the welcome content lettering animation.
jquery.lettering.js Lettering.js is lets us work with letters individually.
jquery.isotope.js jQuery Isotope.js is used for the dynamic portfolio filtering.
script.js The main javascript file used in AvaLance, where the functions are defined.
waypoints.min.js jQuery Waypoints let us execute a function whenever you scroll to an element.
waypoints-sticky.min.js jQuery Waypoints Sticky plugin used to create the sticky navigation bar.

Template Editing Customizing template settings

Heads up! To edit files in this template you need a text editor software installed in your computer.

Below are some of popular text editor software out there:

  1. Sublime Text (Cross Platform)
  2. Notepad++ (Windows)
  3. TextMate (Mac)

How To Edit Colors

Open the css color skin file you'll want to edit (e.g. cold.css), and then do the following

/* Section Titles */
.section-title {color: #yourColor;}

How To Change Icons

Choose the wished one from the icon reference, then simply replace the sample icon's name (e.g. icon-twitter) in it's class to the chosen one.

<ul class="social-networks"> 
	<li><a href="#"><i class="icon icon-twitter"></i></a></li> 
	<li><a href="#"><i class="icon icon-facebook"></i></a></li>					 
</ul>

How To Change Portfolio Filters

Locate the filters div in the HTML file, then change the attributes to your desired filters in the data-option-value="..." element.

<li><a href="#filter" data-option-value=".graphicdesign"> 
	<i class="icon icon-pencil"></i><span class="filter-title">Graphic Design</span> 
</a></li>

How To Apply Portfolio Filters

Locate the portfolio-wrapper div, then add the specific filter's class to each <a class="..."> element. Note: You can add several filters to a portfolio item.

<!-- Portfolio Item --> 
<a href="portfolio-details.html" class="graphicdesign">				 
	<figure class="one-third column portfolio-item"> 
		<div class="picture"><img src="img/portfolio/portfolio_01s.jpg" alt=""/></div> 
		<figcaption class="portfolio-item-title"><span>+</span>Lorem ipsum dolor sit amet</figcaption>	 
	</figure> 
</a>

How To Use Video Background

To use the home-video background version, you have to convert your videos to the HTML5 formats .ogv, .webm, .mp4 then upload them in the "video" folder, with the filename "bg_video". Miro Converter could be a good start to convert your videos. Don't forget to create a fallback image for the video video_bg.jpg, which will be shown on small devices.

How To Apply Background Patterns

At the default home version, locate the div with the id pattern-overlay and change it's class to the desired pattern version.

<div id="pattern-overlay" class="pattern-02">

How To Use The Maps

Open the script.js file, and locate myMap on line 326, and change the coordinates to your own ones. You can get your coordinates here.

myMap(40.81992, -73.90683, 'map1');
myMap(40.80287, -73.95361, 'map2');

How To Use Tablet Slider

To change or add images to the tablet slider in the portfolio details page, just upload your images in the "img/portfolio-details" folder, and link them.

<ul id="tablet-slider"> 
	<li class="slide"> 
		<img src="img/portfolio-details/tablet_slide_01.jpg" alt="" />				 
	</li> 
	<li class="slide"> 
		<img src="img/portfolio-details/tablet_slide_02.jpg" alt="" />		 
	</li>	 
	<li class="slide"> 
		<img src="img/portfolio-details/tablet_slide_03.jpg" alt="" />					 
	</li>						 
</ul>

Installation How to Install the Template

Install Theme

Here's a brief information about how to install Omnibiz from scratch

1.1) Organize Your Files

Pick your color scheme and layout first, and the extract zip file and then upload the files to your online host. Remember, you have to rename the chosen home layout to index.html when you uploaded it.

1.2) Customize the settings

Customize the template to your liking according to the instructions above. Feel free to contact me if you have any further questions.

1.3) Refresh your browser.

In order to see whether the editing has been done or not, you need to refresh your browser. If you don't see any changes then the browser's cache need to be cleared.
